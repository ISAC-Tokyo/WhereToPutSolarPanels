# How to setup Development environment

## Requirement

- Node.js >= 0.8.23
- npm modules
 - bower
 - yo
 - grunt-cli

## Install node modules

```
npm install -g yo bower grunt-cli
```

## Init working direcotry

```
cd web_client
bower install
```

## Add JavaScript Library

```
# For example
bower install angular
```
