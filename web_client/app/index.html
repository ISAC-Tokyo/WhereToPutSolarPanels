Where to put solar panels?

The aim of this application is to find the best spots where to install solar panels. How is it possible? The idea here is simple: Finding areas with fewer clouds! This system detects areas with less cloud by analyzing historical satellite data over the 2001-2012 period, and by evaluating expected solar power generation. Spots with the best sun exposition in the past can then be used to model return on investment and to guide decisions on where to put the panels.
